# OneX
![](https://cdn4.iconfinder.com/data/icons/space-and-astronomy-1/800/rocket-128.png)

OneX致力于搭建一套软件项目中经常遇到的一些常见需求,比如权限管理、消息管理、日志管理等。     
结合代码生成器工具,减轻开发人员的繁琐开发工作。
